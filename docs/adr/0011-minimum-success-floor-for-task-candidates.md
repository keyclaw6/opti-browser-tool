# ADR-0011: Minimum 40% success floor for task candidates

- Status: Superseded
- Date opened: 2026-07-11
- Date superseded: 2026-07-11
- Decision owner: project owner
- Superseded by: [ADR-0012](0012-reference-success-band-35-to-70.md)

## Context

This ADR recorded a 40% minimum after interpreting an earlier project-owner message. The project owner then explicitly corrected that interpretation: the required range is **35% to 70%, inclusive**, with a 35% floor rather than 40%.

## Historical decision

The superseded rule screened source families and final tasks at a minimum of 40%, while leaving the upper saturation threshold open. It correctly distinguished benchmark aggregates from task-level evidence, but its numeric floor was wrong.

## Supersession reason

ADR-0012 is the active rule. This file remains to make the correction auditable and to prevent future agents from repeating or hiding the mistake.
