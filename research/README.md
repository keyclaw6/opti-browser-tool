# Research workspace

This directory contains sourced research outputs used to inform decisions before major implementation choices are accepted.

- `harnesses/`: browser-agent repository reviews, architecture comparisons, and hypotheses. This workstream is not yet complete.
- `benchmarks/`: completed source-level benchmark research, the exact 140-task candidate batch, provenance, and pending task-level validation plans.
- `judges/`: verifier designs, judge studies, calibration cases, and error analyses. This workstream is not yet complete.

Research documents must distinguish observed facts, inferences, hypotheses, proposals, and accepted decisions. A promising implementation pattern or published score is not automatically a project decision. The authoritative decision status is in `docs/DECISION_REGISTER.md`.

Use `docs/README.md` as the main reading map and `docs/AGENT_HANDOFF.md` for the current evidence boundary.
